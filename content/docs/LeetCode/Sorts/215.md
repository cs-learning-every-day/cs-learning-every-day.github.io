使用优先队列

```java
    // Time Complexity: O(nlgn)
    // Space Complexity: O(k)
    public int findKthLargest(int[] nums, int k) {
        PriorityQueue<Integer> q = new PriorityQueue<>();
        for (int num : nums) {
            if (q.size() == k) {
                if (num > q.peek()) {
                    q.poll();
                    q.offer(num);
                }
            } else {
                q.offer(num);
            }
        }
        return q.peek();
    }
```
