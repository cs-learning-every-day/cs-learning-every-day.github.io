贪心：最小值买入，最大值卖出

```c++
    int maxProfit(vector<int>& prices) {
        int res = 0;
        int cur = prices[0];
        for (int p : prices) {
            if (p < cur) {
                cur = p;
            } else {
                res = std::max(res, p - cur);
            }
        }
        return res;
    }
```
