```java
    public int translateNum(int num) {
        return dfs(String.valueOf(num), 0);
    }

    private int dfs(String str, int start) {
        if (start >= str.length()) return 1;
        int res = 0;
        
        int tmp = str.charAt(start) - '0';

        if (isValid(tmp)) {
            res = dfs(str, start + 1);
        }

        if (start < str.length() - 1) {
            // 06
            if (tmp == 0) return res;

            tmp = tmp * 10 + (str.charAt(start + 1) - '0');
            if (isValid(tmp)) {
                res += dfs(str, start + 2);
            }
        }
        return res;
    }

    private boolean isValid(int val) {
        return 0 <= val && val <= 25;
    }
```

动态规划
```java
    public int translateNum(int num) {
        String str = String.valueOf(num);
        int n = str.length();
        // 以i为结尾 不同翻译数
        int[] dp = new int[n + 1];
        dp[0] = 1;
        dp[1] = 1;

        for (int i = 2; i <= n; i++) {
            dp[i] = dp[i - 1];
            int val = (str.charAt(i - 2) - '0') * 10 + (str.charAt(i - 1) - '0');
            if (10 <= val && val <= 25) {
                dp[i] = (dp[i - 2] + dp[i - 1]);
            }
        }

        return dp[n];
    }
```
