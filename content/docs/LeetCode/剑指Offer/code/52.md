注意没有公共节点可以看成，最后一个空节点为公共的
```c++
    ListNode *getIntersectionNode(ListNode *headA, ListNode *headB) {
        auto p = headA, q = headB;
        while (p != q) {
            p = (p == nullptr ? headB : p->next);
            q = (q == nullptr ? headA : q->next);
        }
        return p;
    }
```
