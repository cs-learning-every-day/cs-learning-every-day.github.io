DP

dp[i][j] 表示能到i行j列不同路径个数

初始化第一行和和第一列为1
```java
    public int uniquePaths(int m, int n) {
        int[][] f = new int[m][n];
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (i == 0 || j == 0) {
                    f[i][j] = 1;
                } else {
                    f[i][j] = f[i - 1][j] + f[i][j - 1];
                }
            }
        }

        return f[m - 1][n - 1];
    }
```

DFS+记忆化
```java
class Solution {
    public int uniquePaths(int m, int n) {
        int[][] memo =  new int[m][n];

        return dfs(0, 0, m, n, memo);
    }

    private int dfs(int i, int j, int m, int n, int[][] memo) {
        if (i >= m || j >= n || i < 0 || j < 0) {
            return 0;
        }

        if (memo[i][j] != 0) {
            return memo[i][j];
        }

        if (i == m - 1 && j == n - 1) {
            return 1;
        }

        int res = dfs(i + 1, j, m, n, memo) + dfs(i, j + 1, m, n, memo);
        memo[i][j] = res;
        return res;
    }
}
```
